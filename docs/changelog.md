# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/).

---

## [1.1.0]

### Added

* Technical documentation.
* Data dictionary.
* Project roadmap.
* Changelog.
* Type hints across the codebase.
* Docstrings for project modules and functions.
* Structured logging.
* Error handling throughout the ETL pipeline.

---

## [1.0.0]

### Added

* Initial ETL pipeline for SECOP II public contracts.
* Data extraction from the SECOP II public API.
* Data transformation using Pandas.
* Star schema data warehouse model.
* PostgreSQL as the analytical database.
* Docker support for local deployment.
* Initial project documentation.
* Power BI dashboards for data visualization.
